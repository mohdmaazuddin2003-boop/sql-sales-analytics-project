# Sales Analytics SQL Project

## Overview

This project analyzes sales data using SQL Server.

## Features

* Monthly revenue analysis
* Top selling products & categories
* Customer spending analysis
* Sales summary view

## Tools Used

* SQL Server (SSMS)
* Excel (for reporting)

## Key Queries

* Revenue by month
* Top customers
* Category performance

## Outcome

Built a complete analytics system using SQL with real-world business queries.
